# Longitudinal Dynamic Causal Modeling Project

Longitudinal fMRI analysis project investigating changes in brain connectivity during cognitive training using Dynamic Causal Modeling (DCM).

This repository contains the complete analysis pipeline, supporting materials, conference presentations, and manuscript developed as part of the Brainchild project.

## Project Overview

The goal of this project was to investigate how effective connectivity between brain regions changes over time as participants acquire new cognitive skills.

The analysis combines:

* Longitudinal fMRI data
* Dynamic Causal Modeling (DCM)
* Hierarchical Bayesian modeling (PEB)
* MATLAB-based processing pipelines
* Automated preprocessing and analysis workflows

The project follows participants across multiple measurement occasions, allowing investigation of how brain networks reorganize during learning and training.

## Repository Contents

### Analysis Pipeline

The `Longitudinal_Pipeline` folder contains MATLAB scripts used to process and analyze the imaging data, including:

* Data preprocessing
* Motion quality control
* Timing file preparation
* First-level model specification
* DCM model estimation
* Parametric Empirical Bayes (PEB) group analysis
* Longitudinal model comparison

The pipeline was designed to support reproducible analysis of large longitudinal neuroimaging datasets.

## Research Significance

Understanding how brain networks adapt during learning is a central question in cognitive neuroscience.

Rather than focusing only on where activity occurs in the brain, this project investigates how communication between brain regions changes over time, providing insight into the neural mechanisms underlying skill acquisition and cognitive development.
